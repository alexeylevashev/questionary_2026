# Legacy-скрипты

Оригинальные скрипты, полученные через ChatGPT. Используются как референс для переписывания.
Запускались последовательно из папки с данными.

## Порядок запуска (старый)

1. `1_Filter.py` — фильтрация анкет по geojson-территориям
2. `2_Status.py` — группировка по соцстатусам  
3. `3_OD.py` — OD-матрицы и транспорт
4. `4_EVA.py` — часы, длительности, BW-модель

## Известные проблемы

- Дублирование кода между файлами (normalize_excel_text, choose_input_xlsx и т.д.)
- Хрупкий парсинг координат
- bare except при чтении xlsx
- Работают только из корня папки с данными
- Нет GIS-экспорта
- Смешаны xlsxwriter и openpyxl
