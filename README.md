# Обработка анкет транспортных обследований

Консольный инструмент для анализа данных транспортных обследований населения.

## Возможности

- **Фильтрация** анкет по территориям (geojson) с автокоррекцией координат
- **Группировка** по социальным статусам и транспорту
- **OD-матрицы** (Origin-Destination) с распределением по парам и транспорту
- **EVA-анализ**: распределение по часам, длительности, BW-модель
- **GIS-экспорт**: GeoPackage, GeoJSON, Shapefile, OD-линии
- **QGIS-проект**: готовый .qgz с настроенными слоями

## Установка

```bash
pip install -r requirements.txt
```

## Подготовка данных

Положить в папку `data/`:
- `анкеты.xlsx` — база анкет
- `границы_МО.geojson` — полигоны территорий
- `Пары.xlsx` — справочник пунктов
- `Соцгруппы.xlsx` — справочник соцгрупп
- `Транспорт.xlsx` — справочник транспорта

## Запуск

```bash
# Полный пайплайн
python -m src.cli

# Отдельные этапы
python -m src.cli filter
python -m src.cli status
python -m src.cli od
python -m src.cli eva
```

Результаты сохраняются в `output/`.

## Настройка

Параметры в `config.yaml`: пути, столбцы, параметры BW-модели, форматы экспорта.
